============
Installation
============


Jiml can be installed from PyPI with tools like ``pip``::

    $ pip install jiml

or latest stable version from `GitHub <https://github.com/vmstarchenko/jiml>`_::

    $ git clone git://github.com/vmstarchenko/jiml.git
    $ cd jiml

Requirements
------------

* **Python**: 3.6, 3.7, 3.8, 3.9
* **Jinja**
* **PyYAML**
* **jsonschema**
* **jinja-vanish**
